# South Shore Harbor Marketplace

Website for **South Shore Harbor Marketplace**: waterfront events, private
dining, and market experiences in New Orleans.

**Live site:** https://roms504318.github.io/sshmp/

## Structure

- `index.html`: the site (single file, no build step)
- `assets/`: optimized web images (logos, plans, renderings)
- `Graphics/`: source graphics package
- `outputs/archive/`: earlier versions

## Run locally

Open `index.html` in a browser, or serve the folder with any static server.
