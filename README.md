# South Shore Harbor Marketplace — Website Demo

Concept website for **South Shore Harbor Marketplace**: waterfront events, private
dining, and market experiences in New Orleans.

**Live demo:** https://roms504318.github.io/sshmp/

## What this shows

- Light, brand-driven site built on the owner's graphics package (The Public Market
  logotype, Bar & Grill badge, watercolor concept renderings)
- Interactive first-floor plan with space-by-space status
- The **Private Room** as a bookable space — three layout options (Open Floor,
  Meeting, Banquet) with capacities and a request-to-book flow demonstrating
  inquiries, deposits, and reservations
- Designed placeholder pages for spaces still in development (Second Floor,
  Public Market vendor leasing, outdoor grounds, Bar & Grill)

## Structure

- `index.html` — the entire site (single file, no build step)
- `assets/` — optimized web images (logos, plans, renderings)
- `Graphics/` — source graphics package
- `outputs/archive/` — earlier concept versions

## Run locally

Open `index.html` in a browser, or serve the folder with any static server.
